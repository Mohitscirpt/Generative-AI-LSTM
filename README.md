# LSTM Text Generation using TensorFlow

## Project Overview
This project implements a text generation model using LSTM (Long Short-Term Memory) with TensorFlow/Keras.

## Dataset
Shakespeare Complete Works (.txt)

## Technologies
- Python
- TensorFlow
- NumPy
- Matplotlib

## Features
- Text preprocessing
- Tokenization
- Sequence generation
- LSTM model
- Text generation

pip install -r requirements.txt

jupyter notebook
